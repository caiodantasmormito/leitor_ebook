## Como Usar

   ```
   - Clone esse repositório:
   $ git clone https://github.com/caiodantasmormito/leitor_ebook.git

   - Entre no diretório:
   $ cd flutter leitor_ebook

   - Instale as dependências:
   $ flutter pub get

   - Inicie o app: 
   $ flutter run
  
   ```
